웹 브라우저의 종류

Internet explorer
마이크로소프트에서 만든 브라우저로 과거 국내 시장에 가장 많이 사용. 현재 사망.
Fire Fox
Mozilla 재단에서 오픈 소스로 만든 것. w3c의 웹 표준을 선도했다.
Chrome
구글에서 만든 브라우저. 좋은 디버거를 갖추고 있어서 디버깅에 많이 활용됨.

예제

Ajax
Ajax(Asynchronous JavaScript and XML, 에이잭스)는 비동기적인 웹 애플리케이션의 제작을 위해 아래와 같은 조합을 이용하는 웹 개발 기법이다.
vue
웹 애플리케이션의 사용자 인터페이스를 만들기 위해 사용하는 오픈 소스 프로그레시브 자바스크립트 프레임워크이다.
react
자바스크립트 라이브러리의 하나로서[2] 사용자 인터페이스를 만들기 위해 사용된다. 페이스북과 개별 개발자 및 기업들 공동체에 의해 유지보수됨.
spring framework
자바 플랫폼을 위한 오픈 소스 애플리케이션 프레임워크로서 간단히 스프링(Spring)이라고도 한다
